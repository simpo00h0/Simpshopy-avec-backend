# Module Media

## Objectif

Bibliothèque média centralisée pour les images de la boutique. Les fichiers sont hébergés sur ImageKit (CDN) pour un rendu rapide.

## Responsabilités

- Upload d'images vers ImageKit et enregistrement en base
- Liste des médias par boutique
- Suppression de médias
- Réutilisation des images (produits, pages, etc.)
